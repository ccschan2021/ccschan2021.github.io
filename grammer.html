文法:

第１課
  • ［名詞A］は［名詞B］です。
    • 例：      は（日經の先生）です。
  •       は      じゃありません。（では）
  •       は［      ／だれ／どなた］ですか。
  •       も      です。（都/也是）
第２課
  • ［これ／それ／あれ／      ］は［物件］です。
  •       は      ですか、      ですか。
  •       は［（性質／何）の物件］です。
  • ⋯⋯［（所屬／だれ）の物件］です。
  • ⋯⋯［所屬の］です。（省略物件）
  • ［この      ／その      ／あの      ］は（所屬の）です
第３課
  • ［ここ／そこ／あそこ／      ］は［場所／ばしょ］です。
  • ［場所／人物／物件／      ］は［どこ／どちら、ここ／こちら、そこ／そちら、あそこ／あちら］です。
  • ［これ／それ／あれ／      ］はどこの［物件］ですか。［出處：如國家出產、公司出品不可用どちら］
  • ［この      ／その      ／あの      ］はいくらですか。
第４課
  • ［      は］今      時      分です。／［      は］今何時ですか。
  •       は      と      です。
  •       は      から      まで［です／動詞］。
  •       は時間に［動詞］。（時間／日月／節日要加に）
第５課
  •       は［場所］へ［移動動詞：行きます／来ます／帰ります］か。
  •       は［手段方法で］［場所］へ［移動動詞］。（＊歩いて不用で）
  •       は［人物と］［場所］へ［移動動詞］。（＊一人で不用と）
  • 例：だれもいません／なにも食べません。何も〜ませんでした。
第６課
  •       は［受詞］を［動詞］。／      は［何なに］を［動詞］。
  •       は［場所で］［受詞］を［動詞］。／      は［どこで］［受詞］を［動詞］。
  • 〜ますせんか。（邀請）
  • ～ますしょう。
第７課
  •       は［手段方法で］［受詞］を［動詞］。
  • ［語言A］は［～語で］［語言B］です。
  • ［      は］［対象］に［受詞］を［動詞］。（輸出）
  • ［      は］［対象］に［受詞］を［      ました］。（接收/輸入）
  • 問：      はもう［受詞］を［      ました］か。
    • 答：はい、もう［      ました］。（已經）
    • 答：いいえ、まだです。（還未）
第８課
  •       は形容詞です。
  •       はどうですか。
    • 答：＋／＋、−／―：そして（連接詞：而且）
    • 答：＋／―、−／＋：か（連接詞：但是）
  • ［な形容詞］な［名詞］。
  • ［い形容詞］❌［名詞］。
第９課
  •       は［名詞］が好き／嫌い／上手／下手（形容詞）です。
  •       は［名詞：死物／觸不到］がわかります／あります。
  • 程度：よくわかります＞大体わかります＞少しわかります＞あまりわかりません＞よくわかりません＞全然わかりません
  • 數量：たくさんあります＞すこしあります＞あまりありません＞全然ありません
  • 原因理由［です／ます］から、結果。
第１０課
  • ［場所］に［死物］があります。／［場所］に［生物］がいます。
  • ［物/人/地］の［位置に］：とvsや
  • ［死物］は［場所］にあります。／［生物］は［場所］にいます。／      はどこにですか。
第１１課
  • ［場所］に［死物］が［量詞］あります。／［場所］に［生物］が［量詞］います。
  • Aを［量詞］とBを［量詞］ください。
  • 全部vs全部で
  • ［期間］に      回［名詞を動詞／場所へ移動動詞／対象に会います］。
  • ［死物］は［場所］に［期間］あります。／［生物］は［場所］に［期間］います。
第１２課
  • A［好/贏］はB［差/輸］より形容詞です。（比較）
  • 問：AとBとどちらが形容詞ですか。
  • 答：A/Bのほうが形容詞です。／どちらも形容詞です。／どちらもすきじゃありません。
  • ［範圍］で［名詞］がいちばん形容詞です。
第１３課
  •       は［名詞］が欲しい（欲しくない）です。
  •       は      を／が食べたいです。
  •       は      を／が      たくないです。
  •       は［場所］へ［目的に］［移動動詞］。
    • 食事します->食事に、します->しに
  • て形
    • Ⅰ、い、ち、り->って
    • Ⅰ、に、み、び->んで
    • Ⅰ、き->いて、ぎ->いで
    • Ⅰ、し->して
    • Ⅱ、ます->て
    • Ⅲ、します->して、きます->きて
第１４課
  • 〜てください。
  • 〜ますしょうか。
  • 〜ています。（現在進行式，否定：〜ていません）
第１５課
  • 〜てもいいですか。（可唔可以?）
  • 〜てはいけません。（禁止）
  • ～ています。（狀態、長時間）
    • 知ります（要知道）、知っています（知道）、知りません（唔知）
    • 住みます（要住）、住んでいます（住在）、住んでいません（不住）
    • 結婚します（要結婚）、結婚しています（已結婚）、結婚していません（未婚）
    • 持ちます（要拿）、持っています（擁有）、持っていません（無擁有）
第１６課
  • 〜て、〜て、［〜たいです／〜ました／〜ます。］
  • 〜てから（先）、〜（後）。
  • （大主語）は（小主語）が形容詞です。
  •       は［い：〜い->くて］〔な／名：です〕、〜です。（全部優點、全部缺點）
  • ない形
    • Ⅰ、あ段＋ない
    • Ⅱ、ます->ない
    • Ⅲ、します->しない、きます->こない
第１７課
  • ない形でください。
  • 〜ないければたりません。／〜ないといけません（必須）
  • 〜ないくてもいいです。（唔做⋯⋯都可以）
  • 辞書形
    • Ⅰ、う段（買う、書く）
    • Ⅱ、ます->る
    • Ⅲ、します->する、きます->くる
第１８課
  •       は［場所で／手段方法で］［名詞／辞書形こと］ができます。（能夠）
  • ［人物の］趣味は［名詞／辞書形こと／何］です。
  • ［辞書形／名詞の／期間］まえに、〜。
  • た形
    • Ⅰ、い、ち、り->った
    • Ⅰ、に、み、び->んだ
    • Ⅰ、き->いた、ぎ->いだ
    • Ⅰ、し->した
    • Ⅱ、ます->た
    • Ⅲ、します->した、きます->きた
第１９課
  •       は［受詞を／場所へ］〜たことがあります。（做過的一件事）
  • 〜たり、〜たり｛しなければなりません、します、しました、したいです。｝＝〜や〜など
  • ［い］〜く［な／名］に＋なります。（變得）
    • いい->よくなります（變好）
第２０課
  • 動
    • 〜ます->辞書形
    • 〜ません->ない形
    • 〜ました->た形
    • 〜ませんでした->なかった
  • い
    • 寒いです
    • 寒くないです
    • 寒かったです
    • 寒くなかったです
  • な／名
    • 〜です->〜だ
    • 〜じゃありません->〜じゃない
    • 〜でした->〜だった
    • 〜じゃありませんでした->〜じゃなかった
第２１課
  • 普通形と思います。
  • ［      は］〔普通形／語句〕と言います。
  • ［動／い］普通形［な／名］普通形（だ）でしょう？（⋯⋯吧？唔肯定的時候）
第２２課
  • 修飾名詞
    • ［動／い］普通形［な］な［名］の＋名詞
    • Aは（修飾）Bです。
    • （修飾）AはBです。
    • （修飾）受詞を動詞。
第２３課
  • ［動／い］普通形［な］な［名］のとき、〜。（在⋯⋯的時候）
  • 辞書形と（就）、〜。（必然）
第２４課
  • ［      は］対象に受詞をあげます。（出）
  • ［      は］対象に受詞をもらいます。（入）
    •       は［私に］受詞をくれます。／［物は］人がくれます。（給我）
  • ［      は］対象に受詞を〜て（行為）もらいます。（拜托）
  •       は／が［私に］受詞を〜て（行為）くれます。（感謝）
  • ［      は］対象に受詞を〜て（行為）あげます。（不用地位給自己高的人）
第２５課
  • ［もし］〜たら（假定條件：如果⋯⋯的話）
    • ［動］～たら／［い］～いかったら／［な／名］〜だったら
    • ［動］～なかったら／［い］～いくなかったら／［な／名］〜じゃなかったら
  • AたらB（A->B）＝〜てから、〜。
  • 〜ても、〜。（即使⋯⋯也⋯⋯）逆接條件
    • ［動］～て＋も／［い］～いくて＋も／［な／名］でも
    • 〜ないくても（即使不⋯⋯）
第２６課
  • ［動／い］普通形［な／名］普通形（だ->な）んです（口語）／のです（書面）。
    • Ⅰ、原因（因為）／解釋（ん＝から）
    • Ⅱ、詳細要求（どうしましたかvsどうしたんですか）
    • Ⅲ、加以確認（推論）
  • ［動／い］普通形［な／名］普通形（だ->な）んですが（因為）、〜ていただけませんか。（能請你幫我嗎？）
  • ［動／い］普通形［な／名］普通形（だ->な）んですが（因為）、〜たらいいですか。（⋯⋯做才好？）
    • 例：英語を習いたいんですが、どうしたらいいですか。
第２７課
  • 可能形（かのうけい）＝ 辞書形ことができます
    • Ⅰ、え段＋ます（よめます、うたえます、およげます）
    • Ⅱ、ます->られます
    • Ⅲ、できます、こられます
  • 能力動詞 – 不可轉可能形（みえます、きこえます）
  • は對比句（例：小さい鳥や魚は飼えますが、犬や猫は飼いません。）
第２８課
  • 〜ますながら（次）、〜（主）。（動作同步、一邊⋯⋯一邊⋯⋯）
  • ～ています。（習慣）
    • Ⅰ、現在進行式
    • Ⅱ、狀態
    • Ⅲ、習慣
  • 例５：      は［Aも］普通形し、〜。（個人主張、意見）
    • Ⅰ、並列（既⋯⋯又⋯⋯）
    • Ⅱ、複數原因＝から「〜し〜から。」
      • 例：地下鉄は速いし、安いし、地下鉄で行きましょう。
第２９課
  •       が［自動詞て］います。（狀態持續）
    • 例：窓が割れています。
  • 〜てしまいます。〔しまいます->しまいました、しまったんです、しまう（辞書形）〕
    • Ⅰ、完了
      • 例３：シュミットさんが持って来たワインはどこですか。……みんなで飲んでしまいました。
    • Ⅱ、殘念、後悔之意味
      • 例２：電車に傘を忘れてしまいました。
第３０課
  •       に      が［他動て］あります。
    • Ⅰ、他動（動作）->自動（狀態）（あの紙に名前が書いてあります。／お酒が買ってあります。）
    • Ⅱ、強調人為狀態（窓が閉めてあります。）
  • 机の上にメモがおいてあります。->メモは机の上においてあります。（助詞互換）
  • 〜ておきます。
    • Ⅰ、預先準備（事先⋯⋯）（来月の出張ですが、ホテルを予約しておきましょうか。）
    • Ⅱ、回復原貌（はさみを使ったら、元の所に戻しておいてください。）
    • Ⅲ、放置不管（資料を片づけてもいいですか。いいえ、そのままにしておいてください。）
  • その資料はおいておいてください。（おいておいて->おいといて）。
第３１課
  • 意向形（いこうけい）＝「〜ましょう」の普通形
    • Ⅰ、お段＋う（やすもう、のろう）
    • Ⅱ、ます->よう
    • Ⅲ、しよう、こよう
  • 意向形と思っています（自分の意志［希望／打算］）
  • 〜ていません
    • 例：はい、「もう」できました。
      • いいえ、まだです。／いいえ、まだ〜ていません。（<-狀態）
  • 辞書形／ない形［つもり］です。（つもり：名詞、只可用於自己的打算）
  • 辞書形／（名詞の）［よてい］です。（よてい／予定：名詞）
第３２課
  • ［〜た／ない形］ほいがいいです。（建議／忠告：〜比較好）
  • ［動／い］普通形［な／名］普通形（だ）でしょう／だろう。（也許／猜測）
    • 例：合格するでしょうか？きっと合格するでしょう。
  • ［動／い］普通形［な／名］普通形（だ）かもしれません。（可能：低概率）
第３３課
  • 命令形（めいれいけい）
    • Ⅰ、え段（いえ、のめ、かけ、だせ、すわれ）
    • Ⅱ、ます->ろ（みろ）
    • Ⅲ、しろ、こい
  • 禁止形（きんしけい）
    • Ⅰ、辞書形な（のむな、いうな、かくな、すわるな）
    • Ⅱ、辞書形な（たべるな、みるな）
    • Ⅲ、辞書形な（するな、くるな）
  • 「      」と買いてあります。（寫住）／「      」と読みます。（讀作）
  • 「      」は命令形／禁止形／辞書形［という］いみです。（いみ＝意味）
  • 「      」は普通形と言っていました。（A講過⋯⋯）轉達
  • 「      」に普通形と伝えていただけませんか。（幫忙轉達）
    • A->B：佐藤さんにあした休むとつたえていただけませんか。
    • B->C：ミラーさんはあした休むと言っていました。
第３４課
  • 〔〜た／名詞の〕とおりに、〜。（跟着⋯⋯）
  • 〔〜た／名詞の〕あとで（之後）、〜（重點）。
  • 〔〜て／〜ないで〕……。（附帶動作）
  • 例文７：～ないで、〜。（沒有⋯⋯、就⋯⋯）
    • 最近エレベーターに乗らないで、階段を使っているんです。
第３５課
  • 条件形（じょうけんけい）
    • Ⅰ、え段＋ば（会えば、書けば、飲めば）
    • Ⅱ、ます->れば
    • Ⅲ、すれば、くれば
    • い形容詞：〜い＋ければ／〜い＋くなければ
    • な形容詞／名詞：＋なら／＋じゃなければ
    • ない形（Ⅰ、Ⅱ、Ⅲ）：      なければ（会かなければ、食べなければ、しなければ、こなければ）
    • 例：います->いなければ、なければ＝如果沒有的話
  • 〜ばいいですか。（＝〜たらいいですか。）
    • 例：どうしたらいいですか＝どうすればいいですか。
  • 名詞なら、〜。（      がいいですよ。／      がいいと思います。）
第３６課
  • ［（可能形の普通形）／辞書形（非意志）／ない形］ように（為了）、～。（點做）
    • 非意志無可能形：わかります、みえます、きこえます、なります、あります、できます、治ります
  • ［（可能形の普通形）／辞書形（非意志）／ないく］ようになります／なりたいです。（習慣／能力轉變：を->が）
  • 辞書形／ない形ようにしています。（刻意做／不做）
  • 例文７：ようにしてください（目標性、間接的請）
    • 例：絶対に遅れないようにしてください。
第３７課
  • 受身動詞／受身形（うけみけい）
    • Ⅰ、あ段＋れます／れる（のまれる、はなされる、いわれる）
    • Ⅱ、ます->られます／られる
    • Ⅲ、します->されます／される、きます->こられます／こられる
  • ［私は］「対象」に受身形。
  • 迷惑受身形（困擾麻煩時候）
    • ［私は］「対象」に「持っている物／１体の一部」を受身形。
  • 客觀受身形（公告說明／歷史法律／發見發明）
    • 〔事／物〕が／は受身形。
  • 例文５：原料（->から）、材料（->で）
    • この家は木でつくられました。
  • 例文６：〜ています（狀態）
    • 香港のやちんは高いと言われています。
第３８課
  • （辞書形の）は形容詞です。
    • テニスをするのはおもしろいです。テニスをみるのはおもしろいです。
  • （辞書形の）が形容詞です。（４種用が：喜好、能力、はやい、おそい）
    • 香港人は歩くのがはやいです。
  • （普通形の）を（忘れました／知っています）。
  • ［動／い］普通形［な／名］普通形（だ->な）のは名詞です。
    • 父は北海道で生まれました->父が生まれたのは北海道です。
第３９課
  • 原因［動：〜て／〜ないくて］〔い：いくて］〔な／名：で〕、結果。
    • 只限心情及能力動詞的原因
  • 名詞で、結果。（承接結果：已發生不好事情）
  • ［動／い］普通形［な／名］普通形（だ->な）ので（因為）、結果。
    • ので比から客觀，用於長輩上司（請求／辨解）
    • ので只可出現一次，用て／くて／で（例文６）
第４０課
  • 疑問詞［動／い］普通形［な／名］普通形（だ）か、〜。
    • IMCまでどうやって行くか、知っていますか。
  • 疑問詞か、〜。
    • 趣味はなんですか。->趣味はなにか、聞いてください。
  • ［動／い］普通形［な／名］普通形（だ）かどうか、〜。
    • 無疑問詞時用：（會不會⋯⋯/有沒有⋯⋯）
    • ⋯⋯ないかどうか（先提好的，例如沒有遺失東西）
  • 〜てみます。（試試看⋯⋯）

  • ［私は］対象に受詞を［もらいます／いただきます］。

第４２課

第４３課
  • 目のそう（外觀看起來好像、予測「いまにも」「もうすぐ」「これから」…、別人心情）

第４５課
